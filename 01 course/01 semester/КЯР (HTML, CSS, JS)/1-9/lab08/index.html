Немного о CSS-свойствах

color:
Задает цвет текста. Пример: color: red;
font-family:
Определяет семейство шрифтов для текста. Пример: font-family: Arial, sans-serif;
margin:
Устанавливает внешний отступ вокруг элемента. Пример: margin: 10px;
width:
Устанавливает ширину элемента. Пример: width: 300px;
height:
Устанавливает высоту элемента. Пример: height: 200px;
display:
Определяет, как элемент отображается в документе. Пример: display: block;
text-decoration:
Определяет стиль украшения текста, такого как подчеркивание. Пример: text-decoration: underline;
